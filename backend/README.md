# Phishing Awareness Simulator Backend

This is the backend for the Phishing Awareness Simulator. It uses Node.js, Express, and TypeScript.

## Getting Started

1. Install dependencies: `npm install`
2. Run the server: `npm start`
3. Access the API at `http://localhost:3000/api`

## Development

- Build the project with `npm run build`.
- Add more API features as needed following the architecture.
